# План: Печатное меню ресторана (A3)

## Цель
Сгенерировать два отдельных PDF (лицевая и оборотная сторона) для типографии с направляющими и удобным экспортом.

## ✅ Большой шаг 1 — Визуальные направляющие (CSS)

- **1.1** ✅ На `#page1` и `#page2` добавить три контура (CSS):
  - Внешняя сплошная линия (чёрная 0.5px): `422×299 мм` — дообрезной формат
  - Линия реза (пунктир, серый 0.5px): с отступом 1 мм внутрь (`420×297 мм`)
  - Safety margin (тонкая серая, 0.3px): с отступом 4 мм от края (`414×291 мм`)
- **1.2** ✅ Все линии `display: none` при `@media print`

## ✅ Большой шаг 2 — Панель управления (fixed)

- **2.1** ✅ Панель `.no-print` с `position: fixed; top: 8px; left: 8px; z-index: 9999`
  - Кнопка «Скачать лицевую» (page1)
  - Кнопка «Скачать оборотную» (page2)
- **2.2** ✅ Логика кнопок: скрыть другую страницу + `.page-break` → `window.print()` → восстановить
  - Пользователь в диалоге печати выбирает «Сохранить в PDF», A3, без полей, 100%

## ✅ Большой шаг 3 — Node.js скрипт (автоматизированный экспорт)

- **3.1** ✅ Инициализировать `tools/` с `npm init`, установить `playwright`
- **3.2** ✅ Написать `tools/export-pdf.js`:
  - Встроенный HTTP-сервер для раздачи статики
  - Headless-браузер, ожидание загрузки
  - Генерация двух PDF: `menu-page1.pdf`, `menu-page2.pdf` (A3, bleed 1 мм)
  - Генерация двух PNG: `menu-page1.png`, `menu-page2.png` (~288 DPI)
- **3.3** ✅ Запуск: `node tools/export-pdf.js`
