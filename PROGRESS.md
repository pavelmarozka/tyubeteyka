# Changelog

## [1.0.0] — 2026-05-26

### Добавлено
- **Цены**: исправлена проверка `item.цена != null` вместо `item.цена ?` — теперь `null` и `0` корректно обрабатываются
- **Примечания**: вывод `примечание` из JSON под названием блюда курсивом (`item-note`)
- **Логотип**: контейнер в первой колонке лицевой стороны, `logo.png` отображается на всю ширину колонки
- **Категории + иконки**: FontAwesome 6.5.1, перед каждой категорией иконка по смыслу
- **Направляющие (guide)**: три контура на каждой странице:
  - Внешняя сплошная — дообрезной формат `422×299 мм`
  - Пунктирная — линия реза с отступом 1 мм
  - Тонкая — safety margin 4 мм
  - Скрываются при `@media print`
- **Панель управления**: `position: fixed` с двумя кнопками «Скачать лицевую» / «Скачать оборотную»
- **Node.js скрипт**: `tools/export-pdf.js` на Playwright:
  - Генерация `menu-page1.pdf` / `menu-page2.pdf` (A3, bleed 1 мм)
  - Генерация `menu-page1.png` / `menu-page2.png` (~288 DPI, 4785×3390 px)

### Изменено
- **Данные**: удалены встроенные `MENU_DATA` из `index.html` (были временно для отладки). Данные только в `menu.json`, загрузка через `fetch`.
- **Кнопки печати**: удалены из HTML/CSS/JS (`print-panel`, `printPage`, `hide-for-print`) — генерация только через `node tools/export-pdf.js`
- **Расположение категорий**: первая страница собирается вручную через `PAGE1_LAYOUT`:
  - Колонка 1: логотип → Первые блюда → Хлеб
  - Колонка 2: Завтрак → Гарниры → Восточная кухня → Соусы
  - Колонка 3: Салаты → Блюда на мангале → Чебуреки → Десерты
  - Вторая страница: все остальные категории (Кофе, Безалкогольные напитки, Фреши, Милкшейки, Смузи)
- **Данные**: встроены в `index.html` компактно (работает без сервера), при наличии сервера — fetch обновляет из `menu.json`
- **CSS**: добавлен `.menu-col` для вертикальной укладки внутри колонок
- **Символ валюты**: `₽` заменён на `р` для корректного отображения при печати в PDF
