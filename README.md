# Ресторанное меню для печати (А3)

## 📋 Описание проекта
Это проект вёрстки ресторанного меню для последующей печати на бумаге формата А3 (420×297 мм). 

**Реальный размер макета:** 422×299 мм (без bleed) — безопасная зона для печати.

## 🚀 Как запустить
1. Откройте файл `index.html` в любом современном браузере (Chrome, Firefox, Safari)
2. Для просмотра в натуральную величину используйте масштаб 100%

## 📤 Экспорт в PDF/Изображения

### Через браузер (Chrome/Edge):
1. Откройте `index.html` в браузере
2. Нажмите `Ctrl+P` (или `Cmd+P` на Mac)
3. В настройках печати:
   - **Целевой принтер:** "Сохранить как PDF"
   - **Размер бумаги:** Пользовательский (422×299 мм)
   - **Поля:** Нет
   - **Масштаб:** 100%
   - **Фон:** Включить "Фоновая графика"

### Через Puppeteer (Node.js):
```bash
npm install puppeteer
node export-pdf.js
```
Скрипт экспортирует PDF с точными размерами 422×299 мм.

### Через WeasyPrint (Python):
```bash
pip install weasyprint
weasyprint index.html menu.pdf
```

### Экспорт в PNG/JPG/TIFF (300 dpi):
Для получения растровых изображений с разрешением 300 dpi:

**Через Photoshop/Illustrator:**
1. Импортируйте PDF
2. Установите разрешение 300 dpi
3. Экспортируйте в нужный формат

**Через ImageMagick:**
```bash
convert -density 300 menu.pdf menu.png
```

**Важно:** Все тексты остаются векторными при экспорте в PDF, что обеспечивает идеальное качество печати.

## 🛠 Используемые технологии
- **HTML5** — семантическая разметка
- **CSS3** — стилизация
- **CSS Grid** — для колонок и сеток
- **CSS Flexbox** — для выравнивания элементов
- **Google Fonts** — шрифты с поддержкой кириллицы

## ✏️ Как заменить данные

### Изменение цен/названий блюд:
Откройте `index.html` и найдите соответствующий блок. Например:
```html
<li class="menu-item">
  <span class="item-name">Эспрессо</span>
  <span class="item-volume">30 мл</span>
  <span class="item-price">150 ₽</span>
</li>
```
Измените текст внутри тегов.

### Добавление нового блюда:
Скопируйте существующий элемент `<li class="menu-item">` и измените данные.

### Изменение стилей:
Все стили находятся в файле `style.css`. Основные классы:
- `.menu-section` — раздел меню
- `.menu-item` — отдельное блюдо
- `.item-name` — название блюда
- `.item-volume` — объём/вес
- `.item-price` — цена

## 🎨 Цветовая схема
- Текст: #000 (чистый чёрный)
- Фон: #fff (чистый белый)
- При подготовке к печати конвертируйте в CMYK самостоятельно

## 📐 Размеры и отступы
- Контейнер: 422×299 мм
- Внутренние отступы (padding): 10–15 мм
- Внешние отступы (margin): 0
- Разрыв страниц: `page-break-after: always` между сторонами

## ⚠️ Важно для типографии
- Не использовать таблицы (`<table>`) — только Grid/Flexbox
- Все изображения должны быть минимум 300 dpi
- Bleed (+3–5 мм) добавляется отдельно при подготовке к печати
