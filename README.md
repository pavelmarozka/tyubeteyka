# Ресторанное меню для печати А3

Веб-версия ресторанного меню, оптимизированная для печати на А3 (420×297 мм) с безопасной зоной **422×299 мм**.

## Быстрый старт

### Запуск веб-сервера (рекомендуется)

Данные меню загружаются из `menu.json`. Для автоматической загрузки при изменении файла откройте страницу через сервер:

```bash
python3 -m http.server 8080
```

Откройте в браузере: http://localhost:8000

### Просмотр без сервера

`index.html` можно открыть напрямую в браузере (данные встроены в HTML, работают без сервера), но изменения в `menu.json` не подтянутся.

## Структура

```
index.html            — страница меню (данные встроены)
style.css             — стили
menu.json             — данные меню (для редактирования)
logo.png              — логотип
tools/
  export-pdf.js       — скрипт генерации PDF/PNG
  package.json        — зависимости
```

## Редактирование меню

Измените `menu.json`. При открытии через веб-сервер данные обновятся автоматически после перезагрузки страницы.

Формат:
```json
{
  "категория": "Кофе",
  "меню": [
    {
      "наименование": "Эспрессо",
      "объём": "1 порция",
      "цена": 180,
      "примечание": ""
    }
  ]
}
```

## Расположение на страницах

Настройка в `index.html`, переменные `PAGE1_LAYOUT` и `PAGE2_LAYOUT`:

**Первая страница (лицевая):**
- Колонка 1: логотип → Первые блюда → Десерты
- Колонка 2: Завтрак → Гарниры → Восточная кухня → Соусы
- Колонка 3: Салаты → Блюда на мангале → Чебуреки → Хлеб

**Вторая страница (оборотная):**
- Колонка 1: логотип → Кофе
- Колонка 2: Лимонады → Смузи
- Колонка 3: Безалкогольные напитки → Фреши

## Экспорт через Node.js (рекомендуется для типографии)

Установите зависимости и запустите генерацию PDF/PNG:

```bash
npm install --prefix tools
node tools/export-pdf.js
```

Результат:
- `menu-page1.pdf` / `menu-page2.pdf` — PDF A3 с bleed 1 мм
- `menu-page1.png` / `menu-page2.png` — PNG 288 DPI

## Визуальные направляющие

На странице отображаются три контура (скрываются при печати):
- Сплошная чёрная — край дообрезного формата 422×299 мм
- Пунктир серая — линия реза с отступом 1 мм
- Тонкая серая — safety margin 4 мм

## Технические требования типографии

| Параметр | Значение |
|----------|----------|
| Дообрезной формат | 422×299 мм |
| После реза | 420×297 мм (А3) |
| Bleed | +1 мм с каждой стороны (заложен в макете) |
| Значимые элементы | отступ от края реза 3 мм |
| Рамка | не менее 4 мм толщиной |
| Разрешение PNG | ~288 DPI (4785×3390 px) |
| Цветовая модель | RGB → CMYK (конвертация в Inkscape) |
